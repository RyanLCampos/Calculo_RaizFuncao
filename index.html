Este programa encontra as raízes de uma polinomial de grau N, dada por seus coeficientes. O usuário deve informar o grau da polinomial, o vetor de coeficientes e um valor de precisão ε.

Para inserir corretamente os coeficientes, leve em consideração a seguinte função:

f(x) = x3 - 9x + 3

Insira os coeficientes em ordem de menor expoente para o maior expoente, exemplo:

3,-9,0,1

Sendo assim, esse será o resultado:

3x0 - 9x1 + 0x2 + 1x3
